Use the data file 'all_indicators.rds'. This includes the following indicators
(by country and year: 1960-2016):
   1. Crude Birth Rate (cbr)
   2. Total Fertility Rate (tfr)
   3. Crude Death Rate (cdr)
   4. Life Expectancy at Birth (e0)
   5. Old-Age Dependency Ratio (oadr)
